# Y2_C_SIM

## Compile and run

Makefile is currently just a rough draft, you should probably navigate to your local ***Y2_C_SIM*** folder and use:
```
gcc -Wall TP[number]/main.c -o prog
./prog
```
... replacing `[number]` with the relevant number
